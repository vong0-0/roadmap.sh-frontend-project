# Get Start
## 1. Clone the repository
```
git clone https://github.com/your-username/roadmap.sh-frontend-project.git

cd roadmap.sh-frontend-project/pomodoro-timer
```
## 2. Install dependencies
```
npm install
```
## 3. Run the app locally
```
npm run dev
```
